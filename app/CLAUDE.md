## Project

Personal portfolio for **Rian Farhan** (Baso Rian Farhan Mallanti) — a full-stack and Android developer in Makassar, Indonesia. Vite + React 19 SPA, React Router 7, Tailwind CSS 3, framer-motion, and gsap. Live site: rianfarhan.com.

## Design Context

Strategic design context lives in **[PRODUCT.md](./PRODUCT.md)** — read it before design or UI work. In short:

- **Register:** brand (a portfolio — the design is the product). **Platform:** web.
- **Audience & goal:** recruiters and hiring managers; the job is an always-on professional presence that makes Rian easy to evaluate and contact.
- **Personality:** precise, technical, confident — sharp and technically impressive. The existing terminal / scan-line / monospace world is identity to preserve, not replace.
- **Primary action:** browse the projects; fallback is contact / hire me.
- **Avoid:** the generic dev-portfolio template, over-designed noise, and the AI-generated gradient/glass look.
- **Motion:** deliberately always-on (reduced-motion is intentionally ignored); readability and contrast must survive the motion.

The visual system (tokens, type, components) is documented in **DESIGN.md** when present. The impeccable design skill is set up here — run `/impeccable` to see context-aware next steps.
